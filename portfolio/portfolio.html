Hello!
My name is Sonja Mihajlov and I am from Serbia.

My first experience with Coding started programming in "DOS" long long time ago, in elementary school; it was so boring that I really hated it. Luckily, in highschool I learnt HTML and CSS which was very interesting to play around however at that time I wanted to study something completely different so I didn't stick with this for a long time. My plan is to master JavaScript and then focus on Python and I am looking forward to use all the tricks that I have learned during this course to create some interesting and useful apps.
